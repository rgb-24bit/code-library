* Text style
  + Single line text overflow shows ellipsis.
    #+BEGIN_SRC css
      div {
        overflow: hidden;
        text-overflow: ellipsis;
        white-space: nowrap;
      }
    #+END_SRC

  + Multi-line text overflow showing ellipsis.
    #+BEGIN_SRC css
      div {
        display: -webkit-box;
        -webkit-box-orient: vertical;
        -webkit-line-clamp: 3;
        overflow: hidden;
      }
    #+END_SRC

  + Center the text up and down in the div.
    #+BEGIN_SRC css
      div {
        height: 20px;
        line-height: 20px;
      }
    #+END_SRC
